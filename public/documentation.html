Movie API

Movie API offers developers information about movies, including title, director, genres, and release dates. It also provides endpoints for user registration, managing favorite movies, and updating user information.

Endpoint Summary

GET /movies — Returns a list of all movies

GET /movies/:title — Returns details of a single movie by title

GET /movies/genre/:genreName — Returns information about a genre

GET /movies/director/:directorName — Returns information about a director

POST /users — Registers a new user

PUT /users/:username — Updates user information

POST /users/:username/movies/:movieID — Adds a movie to user's favorites

DELETE /users/:username/movies/:movieID — Removes a movie from user's favorites

DELETE /users/:username — Deregisters a user

Detailed Endpoint Documentation

GET /movies

Description: Returns a list of all movies.

Request Body: None

Response Example:

[ { "title": "Inception", "description": "A dream thief enters people's subconscious.", "genre": { "name": "Sci-Fi", "description": "Science fiction" }, "director": { "name": "Christopher Nolan", "birth": 1970 }, "imageURL": "https://...", "featured": true } ]

GET /movies/:title

Description: Returns data about a single movie by title.

URL Parameters: :title

Response Example:

{ "title": "Inception", "description": "A dream thief enters people's subconscious.", "genre": { "name": "Sci-Fi" }, "director": { "name": "Christopher Nolan" }, "imageURL": "https://...", "featured": true }

GET /movies/genre/:genreName

Description: Returns information about a genre by name.

URL Parameters: :genreName

Response Example:

{ "name": "Thriller", "description": "Suspenseful and exciting films." }

GET /movies/director/:directorName

Description: Returns information about a director.

URL Parameters: :directorName

Response Example:

{ "name": "Christopher Nolan", "bio": "Known for non-linear storytelling and complex plots.", "birth": 1970, "death": null }

POST /users

Description: Registers a new user.

Request Body Example:

{ "username": "newuser", "password": "mypassword", "email": "user@example.com", "birthday": "1990-01-01" }

Response Example:

{ "message": "User successfully registered." }

PUT /users/:username

Description: Updates a user's information.

URL Parameters: :username

Request Body Example:

{ "username": "updatedUser", "email": "updated@example.com" }

Response Example:

{ "message": "User information updated." }

POST /users/:username/movies/:movieID

Description: Adds a movie to user's favorites.

URL Parameters: :username, :movieID

Response Example:

{ "message": "Movie added to favorites." }

DELETE /users/:username/movies/:movieID

Description: Removes a movie from user's favorites.

URL Parameters: :username, :movieID

Response Example:

{ "message": "Movie removed from favorites." }

DELETE /users/:username

Description: Deregisters a user account.

URL Parameters: :username

Response Example:

{ "message": "User account removed." }
